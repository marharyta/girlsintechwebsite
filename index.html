About The Event

Girls in Tech Helsinki goes FROSMO: The Coding Day is a practical workshop on the topics of web development and tools on October 29, 2016 in co-operation with FROSMO, the forerunner of the Single Tag based UI improvement methodology.

This 1-day Workshop offers attendees the chance to finesse existing skills while mingling with peers, forming new relationships and networking with international professionals.

GIT Helsinki goes FROSMO: The Coding Day is open for anyone interested in developing and improving their coding skills. This 6-hour workshop has been designed for beginners with basic knowledge in HTML/CSS/JavaScript languages and will receive assistance from the coding specialists of FROSMO.

The event is completely free and open for both women and men.

The event has capacity for 20 people. The deadline for the application is on October 15, 2016 at 23:59. All the applicants will be notified of the results on October 17, 2016 via e-mail. Please bring your own laptop for this session.
